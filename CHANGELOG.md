# Change Log

All notable changes to the "BrainSync Focus Timer" extension will be documented in this file.

## [0.1.0] - 2026-02-11

### Added
- 30分作業 + 5分休憩のポモドーロタイマー（15-60分でカスタマイズ可能）
- 4セット完了後の長い休憩（15分、カスタマイズ可能）
- ステータスバー表示（残り時間、状態アイコン）
- クイックピックメニューによる操作
- 通知システム（作業完了、休憩完了、脳疲労アラート）
- サウンド再生（作業終了、休憩終了、アラート）
- 統計機能（日次・週次の集計、Webview表示）
- 脳疲労スコア推定（0-45点、5つの指標から算出）
- BrainSync脳疲労診断ページとの連携（4箇所の導線）
- データエクスポート（CSV形式、週/月/全期間）
- VS Code再起動時のタイマー復元
- 設定のカスタマイズ（タイマー時間、通知、サウンド、アラート閾値）
